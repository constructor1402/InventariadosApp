# InventariadosApp

Aplicación móvil desarrollada en **Kotlin** con **Jetpack Compose** y **Firebase**, 
creada para la gestión de equipos topográficos, su asignación, mantenimiento y 
control por parte de administradores y topógrafos.

---

## Características principales

- **Gestión de equipos:** Registro, actualización y eliminación de equipos con información detallada (nombre, serial, estado, etc.).
- **Asignación y devolución:** Permite asignar equipos a obras o topógrafos y registrar la devolución.
- **Módulo de informes:** Genera reportes en formato PDF de los equipos y su estado.
- **Certificados:** Subida y visualización de certificados en PDF almacenados en Firebase Storage.
- **Roles de usuario:** Control de acceso por roles (Administrador, Topógrafo y Consulta).
- **Integración con Firebase:**
  - Firestore (base de datos en tiempo real)
  - Firebase Auth (autenticación de usuarios)
  - Firebase Storage (almacenamiento de archivos)
- **Modo oscuro y diseño moderno:** Implementado con Material 3 y Jetpack Compose.

---

## Tecnologías utilizadas

- **Lenguaje:** Kotlin  
- **Framework UI:** Jetpack Compose  
- **Base de datos:** Firebase Firestore  
- **Almacenamiento:** Firebase Storage  
- **Autenticación:** Firebase Auth  
- **Control de versiones:** Git + GitHub
